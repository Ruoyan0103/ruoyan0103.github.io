From Model to Production

This project demonstrates the journey from a machine learning model to a production-ready application. It is inspired by the book Deep Learning for Coders with Fastai and PyTorch: AI Applications Without a PhD , chapter 2, but adapted to use the PyTorch library.

Dataset

The dataset for this project is sourced via DuckDuckGo.

Architecture

The pretrained resnet18 model was chosen and fine-tuned on this dataset. Training was conducted in HPC interactive mode to leverage GPU acceleration. All code is implemented in a Jupyter Notebook.

Deployment

The model is deployed using Binder and is fully capable of performing inference.
